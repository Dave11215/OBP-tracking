Running sessions

Easy Run + strides — Zone 2

30–40 min conversational (HR ~125–140, cap 145), then 4–6 x 10–15s strides. Keep it genuinely easy.

Quality Run — midweek hard (gated)

5K tempo / threshold. Warm up, finish strong but not to failure, cadence high. Swap to Bike VO2 if legs/HRV need it.

Track intervals — speed (gated)

WU 12–15 min + drills, then 5–6 x 400–800 m at ~5K effort (HR to upper Z4, cap ~172) with generous recovery, CD. Never a time trial.

Long Run — weekend endurance

Easy-to-moderate, progressing ~a mile a week (~4 → 5 → 6 → 6–7). Outdoors on the weekend.

Bike sessions

Zone 2 Bike — easy base, joint-sparing

Conversational Zone 2, HR 120–140 (soft cap 145), 45–90 min. Long outdoor ride on the weekend; carbs + salted fluid past ~75 min.

Bike VO2 — impact-free hard

WU 10–12 min, intervals at ~90–95% max HR (3–4 min on / 3–4 easy), CD. Redline is fine, no impact. Never to failure. This is where the over-threshold work lives.

Prehab routines

Knees + ankle/foot ride with your runs (do them as a quick pre-run activation) and strength days; shoulders/elbows is a daily near-freebie. A few minutes each.
Shoulders / Elbows — daily, ~3–5 min
  • Reverse Tyler Twist · for the right common-flexor tendon, 2 x 12-15
  • Band external + internal rotation · 2 x 12/side
  • Scapular / face pulls · 2 x 12-15
  • Hammer eccentric pronation + supination · slow, 2 x 10/side

Protects the right elbow (partial common-flexor tear + golfer's elbow) and shoulder. Keep sub-painful.

Knees — run + strength days, ~3–5 min
  • Step-ups · controlled, 3 x 12
  • Sit-downs to a bench/box · 3 x 12
  • Reverse lunges · 3 x 12
  • Lateral lunges · 3 x 12 — the frontal-plane work running never trains
  • Star taps · 3 x 12
  • Wall sits · 3 x 30s
  • Eccentric step-downs · add these: highest-value patellar-tendon move, 2 x 8/leg slow

Maintenance for the knee (the strength-before-running gate that closed the recurrence cycle). Cadence toward 165+ is the #1 running-mechanics fix alongside this.

Ankle / Foot — run + strength days, ~5 min (trail-driven)
  • Single-leg balance · 30s/leg; eyes open → closed → on a cushion
  • Band eversion (peroneals) · slow, 2 x 15/leg — resists the ankle-roll
  • Eccentric calf raise · 3s lower, 2-3 x 10/leg (straight + bent knee)
  • Single-leg RDL reach · 2 x 8/leg
  • Knee-to-wall dorsiflexion · 2 x 10/leg
  • Short-foot / toe splay · 5 x 5s hold

Ankle stability for trail. Add hop-and-stick (2 x 5/leg) from week 3. The left ankle is a running abort signal.
